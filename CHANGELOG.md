# Changelog

## 0.1.0 — 2026-07-14

- Initial exact Duel and damage engine.
- Added individual Duel-die identity, Fight values, modifiers and natural overrides.
- Added ordered pool rerolls and explicit both-side reroll order.
- Added standard and split To Wound support.
- Added modifier application to both split stages.
- Added first-stage Might carryover.
- Added Trapped/Prone non-stacking handling.
- Added damage distributions, Wounds and repeated Fate attempts.
- Added Galadhrim pike examples and project audit.
- Added 17 regression tests.
